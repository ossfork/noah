# Noah for Tinkerers

**真正能解决问题的IT支持。** Noah是一款桌面应用，用自然语言诊断和修复电脑问题。描述哪里出了问题，Noah会调查原因、展示方案，一键修复。

不用提工单。不用排队等待。不用搜索错误代码。

## 工作原理

1. **描述问题** — 用你自己的话，不需要专业术语
2. **Noah调查** — 在后台静默运行诊断
3. **Noah展示方案** — 发现了什么问题，打算怎么解决
4. **你点击一个按钮** — Noah完成剩下的工作并确认修复结果

每个操作都有记录。危险操作需要你的明确批准。Noah绝不触碰启动配置、固件、安全软件或系统保护文件。

## Noah能解决的问题

**"网速好慢"** — Noah检查你的网络、DNS和连接状况。找到问题（错误的DNS服务器、过期缓存、连错了网络），然后修复。

**"打印机不打印了"** — Noah检查打印机，找到卡住的任务或崩溃的打印服务，清除队列，重启服务。

**"电脑卡得不行"** — Noah识别什么在占用你的CPU和内存，显示问题进程并终止它。如果是缓存臃肿就清理缓存。

**"这个应用老是崩溃"** — Noah查看日志，识别规律，清除损坏的缓存，让你回到工作状态。

**"这个问题之前也遇到过"** — Noah会记住。它会保存关于你系统的知识——设备信息、之前的修复方案、你的偏好——每次使用都变得更聪明。

## 开始使用

### 下载

前往[Releases](https://github.com/xuy/noah/releases)下载最新版本：
- **macOS** — `.dmg`（Apple Silicon）
- **Windows** — `.msi`或`.exe`安装程序（x64）

> **macOS提示：** Noah尚未获得Apple开发者证书签名。右键点击应用，选择"打开"，再次点击"打开"即可。仅首次需要。

### API密钥

Noah使用Claude（Anthropic出品）来分析问题。你需要一个API密钥：

1. 在[console.anthropic.com](https://console.anthropic.com)获取密钥
2. 在Noah的设置界面粘贴 — 完成

你的密钥保存在本地机器上，仅用于直接与Anthropic API通信。

## 安全性

- **三思而后行** — 始终先运行只读诊断
- **展示方案** — 执行前你可以看到Noah将要做什么
- **标记风险操作** — `rm`、`sudo`、磁盘格式化等命令需要明确批准，附带通俗易懂的解释
- **记录一切** — 每个操作都记录在会话日志中，可以查看和撤销
- **硬性限制** — 启动配置、固件、安全软件、磁盘分区和系统完整性保护永远不可触碰
