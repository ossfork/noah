# Noah for Tinkerers

**本当に問題を解決するITサポート。** Noahは、コンピュータの問題を自然な言葉で診断・解決するデスクトップアプリです。問題を説明するだけで、Noahが調査し、計画を示し、ワンクリックで修正します。

チケット不要。待ち時間なし。エラーコードの検索も不要。

<p align="center"><i>「パソコンが遅い」と言うだけ。Noahが原因を見つけ、解決策を説明し、対処します。</i></p>

## 使い方

1. **問題を説明する** — 自分の言葉で、専門用語は不要
2. **Noahが調査する** — バックグラウンドで診断を実行
3. **Noahが計画を示す** — 何を見つけたか、何をするか
4. **ボタンを1つクリック** — Noahが残りを処理し、修正を確認

すべてのアクションが記録されます。危険な操作には明示的な承認が必要です。Noahはブート設定、ファームウェア、セキュリティソフト、システム保護ファイルには一切触れません。

## Noahが解決する問題

**「インターネットが遅い」** — Noahがネットワーク、DNS、接続を確認。問題を特定（不良DNSサーバー、古いキャッシュ、間違ったネットワーク）して修正します。

**「プリンターが印刷しない」** — Noahがプリンターを確認し、詰まったジョブやクラッシュした印刷サービスを発見、キューをクリアしてサービスを再起動します。

**「パソコンが重い」** — NoahがCPUとメモリを消費しているものを特定し、問題のプロセスを表示して停止します。キャッシュの肥大化が原因ならクリアします。

**「アプリが落ち続ける」** — Noahがログを確認し、パターンを特定、破損したキャッシュをクリアして作業に戻れるようにします。

**「この問題は前にもあった」** — Noahは覚えています。システムについて学んだこと（デバイスの詳細、過去の修正、あなたの好み）を保存し、セッションごとに賢くなります。

## はじめに

### ダウンロード

[Releases](https://github.com/xuy/noah/releases)から最新版をダウンロード：
- **macOS** — `.dmg`（Apple Silicon）
- **Windows** — `.msi`または`.exe`インストーラー（x64）

> **macOS注意：** NoahにはまだApple開発者証明書がありません。アプリを右クリック→「開く」→もう一度「開く」で起動できます。初回のみ必要です。

### APIキー

NoahはClaude（Anthropic社）を使って問題を推論します。APIキーが必要です：

1. [console.anthropic.com](https://console.anthropic.com)でキーを取得
2. Noahのセットアップ画面に貼り付け — 完了

キーはあなたのマシンにローカル保存されます。Anthropic APIとの直接通信にのみ使用されます。

## 安全性

- **行動する前に確認** — 常に読み取り専用の診断を最初に実行
- **計画を表示** — Noahが何をするか、実行前に正確に確認できます
- **危険なアクションを警告** — `rm`、`sudo`、ディスクフォーマットなどは平易な説明付きで明示的な承認が必要
- **すべてを記録** — すべてのアクションがセッションジャーナルに記録され、確認・元に戻すことが可能
- **厳格な制限** — ブート設定、ファームウェア、セキュリティソフト、ディスクパーティション、システム整合性保護は永久に操作対象外
