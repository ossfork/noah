FAQ

Is Noah free?

Yes. Noah for Tinkerers is open-source (AGPL-3.0). You just need your own Anthropic API key for the AI.

What can Noah fix?

Network issues, slow performance, crashing apps, printer trouble, DNS configuration, disk cleanup, and more. If you'd Google it, Noah can probably fix it.

Is it safe?

Noah runs read-only diagnostics first, asks permission before every consequential change, and logs everything. Catastrophic operations — wiping your home folder or keychains, erasing a disk — are blocked outright and can't be approved away.

Does my data leave my computer?

The conversation with Claude goes to the Anthropic API, authenticated with your own key. Your key, session journal, and system data stay on your machine. The only other network calls are a signed update check and an anonymous, opt-out usage ping.

Prefer it fully managed?

onnoah.app is the same idea with nothing to set up — no API key, no configuration. Two parallel tracks to the same place.
